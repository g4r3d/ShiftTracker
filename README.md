# ⚙️ ShiftTracker — Производственный учёт смен

> Настольное приложение для учёта производственных смен на промышленных предприятиях.  
> Учёт плавок, сотрудников и заказчиков с экспортом в Excel.

## 🚀 Установка

1. Перейди на страницу [**Releases**](https://github.com/g4r3d/ShiftTracker/releases)
2. Скачай `ShiftTracker.exe` из последнего релиза
3. Положи файл в любую удобную папку
4. Запусти — при первом старте выбери файл для хранения данных

> ⚡ Никакой дополнительной установки не требуется

## ✨ Возможности

### 📅 Смены
- Календарь с цветовой индикацией  
  🟡 **Жёлтый** — активна одна смена  
  🟢 **Зелёный** — активны обе смены  
- Учёт сотрудников, печей, заказчиков по каждой смене
- Учёт плавок, тиглей и корректировок проб
- Поле для примечаний

### 📊 Отчёты
- Группировка по датам с раскрытием по сменам
- Фильтры по сотруднику, печи и заказчику
- Итоговая статистика: смен / плавок / проб за период
- Экспорт в **Excel** — формулы, динамические колонки по каждой печи, итоги

### 📁 Справочники
- Сотрудники, заказчики, печи
- Для каждой печи задаётся количество тиглей
- Поиск, скрытие, редактирование

### 🎨 Интерфейс
- Тёмная и светлая тема — переключение в реальном времени
- Настройка размера шрифта
- Управление файлом данных (открыть существующий / создать новый)

### 🔒 Безопасность
- Опциональный пароль при входе

### 🔄 Обновления
- Автоматическая проверка новых версий
- Обновление одной кнопкой прямо из программы

---

## 💾 Данные

Все данные хранятся локально в одном `.json` файле на вашем компьютере — смены, сотрудники, печи, заказчики и настройки.  
Файл **никуда не отправляется**. При обновлении программы данные **сохраняются**.
